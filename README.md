# Mintlify Documentation Website

This project is a documentation-style website inspired by the Mintlify platform. It was built using only HTML and CSS with a focus on layout, readability, and structured design.

## Sections Included

The following sections were recreated in this project:

- Navigation bar with logo, links, and action buttons  
- Hero section with heading, description, and email input  
- Documentation preview with sidebar navigation and content cards  
- Trusted companies section with logos  
- Feature section with UI preview  
- Enterprise features  
- Customer stories  
- Call-to-action section  
- Footer with multiple columns  

---

## Fonts Used

The website uses the system font stack:

```
system-ui, -apple-system, Segoe UI, Roboto, sans-serif
```

This keeps the design clean and ensures consistent appearance across different devices.

---

## Colors Used

- Background: `#020617`  
- Card background: `#020817`  
- Borders: `#0f172a`  
- Accent color: `#34d399`  
- Primary text: `#ffffff`  
- Secondary text: `#94a3b8`  

These colors were selected to maintain good contrast and readability while matching the style of a modern documentation website.

---

## Technologies

- HTML5  
- CSS3  

No JavaScript or frameworks were used.
